# Daily Journal App

A JavaScript DOM project that provides a simple digital journal for writing daily thoughts. The application automatically counts words, tracks writing progress, displays the remaining word limit, changes the interface based on writing progress, and saves journal entries using Local Storage so they remain available even after refreshing the page.


## 🚀 Features

- Write journal entries in a clean editor
- Live word counter
- 300-word writing limit
- Dynamic progress bar
- Remaining word counter
- Visual status indicators (Normal, Warning, Success, Danger)
- Auto-resizing textarea
- Automatic saving using Local Storage
- Restore previous journal entry on page reload
- Responsive and modern user interface

## ⚙️ How It Works

1. Users write journal entries inside the textarea.
2. The application counts words in real time.
3. A progress bar visually represents writing progress toward the 300-word limit.
4. Different status colors are displayed depending on the number of words written:
   - Normal
   - Warning
   - Success
   - Danger
5. Journal content is automatically stored using Local Storage.
6. Previously saved entries are restored automatically when the page is reopened.

## ▶️ How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Launch `index.html` in your browser.
4. Start writing your journal.
5. Watch the live word counter and progress bar update automatically.
6. Refresh the page to see your journal restored from Local Storage.
