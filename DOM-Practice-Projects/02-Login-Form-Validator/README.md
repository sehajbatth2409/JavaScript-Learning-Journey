# Login Form Validator

A JavaScript DOM project that validates user login credentials using regular expressions (Regex). The application checks whether the entered email follows a valid format and ensures the password satisfies common security requirements before allowing submission.


## 🚀 Features

- Validate email format using Regular Expressions (Regex)
- Validate password strength
- Display real-time error messages
- Prevent invalid form submission
- Clear form fields after successful validation
- Clean and responsive user interface
- Instant client-side validation without page reload

## 🔒 Validation Rules

### Email Validation

The email must:

- Follow a valid email format
- Contain a username
- Include the `@` symbol
- Include a valid domain name

### Password Validation

The password must contain:

- Minimum 8 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one numeric digit
- At least one special character

## ▶️ How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Launch `index.html` in your browser.
4. Enter an email and password.
5. The application validates the inputs and displays appropriate error messages or a success alert.
