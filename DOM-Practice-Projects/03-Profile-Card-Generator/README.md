# Profile Card Generator

A beginner-friendly JavaScript DOM project that dynamically generates profile cards from user input. Users can enter profile details such as an image URL, name, occupation, and a short description, and instantly create beautifully styled profile cards without refreshing the page.


## 🚀 Features

- Create profile cards dynamically
- Upload profile images using image URLs
- Display user's name, occupation, and about section
- Automatic form reset after card creation
- Responsive card layout using CSS Grid
- Modern and clean user interface
- Instant DOM updates without page reload

## ▶️ How to Run

1. Clone or download this repository.
2. Open the project folder.
3. Launch `index.html` in your browser.
4. Fill in the profile details.
5. Click **Create Profile** to generate a new profile card.
