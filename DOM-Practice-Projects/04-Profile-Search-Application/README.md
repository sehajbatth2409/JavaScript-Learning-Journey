# Profile Search Application

A JavaScript DOM project that displays a collection of user profile cards and allows users to search profiles instantly using a live search feature. The application dynamically filters and updates the displayed profiles based on the user's input without reloading the page.


## 🚀 Features

- Display user profiles dynamically
- Live search functionality
- Filter profiles by name
- Dynamic DOM rendering
- Responsive card layout
- Interactive hover effects
- Instant search results without page refresh
- Clean and modern user interface


## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (DOM Manipulation)

- appendChild()

- setAttribute()
- Template-Based UI Generation


## ⚙️ How It Works

1. A collection of user profiles is stored as an array of JavaScript objects.
2. Each profile contains:
   - Name
   - Profile Image
   - Short Bio
3. The application dynamically creates profile cards using JavaScript.
4. As the user types into the search bar, the application filters profiles based on the entered name.
5. Matching profiles are instantly displayed while non-matching profiles are hidden.

## ▶️ How to Run

1. Clone or download the repository.
2. Open the project folder.
3. Launch `index.html` in your browser.
4. Use the search bar to filter profiles by name.
5. Matching profiles will update instantly as you type.
